Python中常见额数据类型
一. 数值类型数据
python中常用的数值类型：
1.整数  int
2.浮点数(小数) float
3.布尔值 bool类型    True or False 
True  条件成立
False  条件不成立
4.复数 在开发中用不到
扩展知识点 ：复数 complex a = 8 + 9i 
Python中用来区分数据类型的函数 Type 
a =100 
print(type(a))

+ - *（乘）/（除） %（除法取余）//（除法取整） **（求幂）

四. 字符串 ：在python中引号包起来的叫做字符串
a.字符串的定义与转义
1.单引号
2.双引号
3.三引号
注意： 单引号和双引号定义字符串没有任何区别,
三引号定义的字符串的时候可以自由的换行
str1 = '''
123
abc
efg
trtiii
'''
需求：字符串中如果有单引号，则怎么定义
1。外面用双引号包起来
字符串中如果有单引号，也有双引号，则怎么定义
1.三引号包起来
2. 字符串的转义

字符串的转义(可以对有特殊意义的字符进行转义)
反斜杠：\':表示单引号'
       \":表示双引号"
       \n:表示换行
       \t：水平制表符  
3.关闭字符串转义:在字符串前面加一个r
b.字符串的拼接与格式化输出
1.字符串的拼接
方式一：字符串支持加法，直接使用+实现拼接

2.字符串的格式化输出
方式二：
1.format格式化输出
2.字符串F表达式
info = F“今天收到{name}同学，交来{fee}费用，金额{money}，开此收据为凭证!”
方式三.传统%号格式化输出
info = “今天收到%s同学，交来%s费用，金额%s，开此收据为凭证!”%(name,fee,money)
%s : 表示为任意的类型占位
%f:表示浮点数
%d:表示数值占位
